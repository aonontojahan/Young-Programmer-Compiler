# Compiler Project

## Description
This is a simple compiler project designed for young programmers. It includes lexical analysis, parsing, symbol table handling, and basic error detection.

## Features
- Lexical analysis......
- Syntax analysis
- Symbol table management
- Sample input/output demonstration

## Folder Structure
- `src/` → Source code
- `docs/` → Project report and diagrams
- `examples/` → Sample input files

## How to Run
1. Navigate to `src/` folder.
2. Compile the program:
   ```bash
   g++ main.cpp lexer.cpp parser.cpp -o compiler
